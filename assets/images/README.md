# Background Image

Letakkan gambar background Anda di folder ini dengan nama:
- `login_background.jpg` atau
- `login_background.png`

Gambar yang direkomendasikan:
- Resolusi: 1920x1080 atau lebih tinggi
- Format: JPG atau PNG
- Tema: Biru, modern, profesional
- Ukuran file: < 2MB untuk performa optimal
